1. js导入方式
		外部导入
			
		内部现写
			

	2. 基本语法
		注释:
			// 单行
			/* 多行 */

		每一条js命令以 分号为结束符

		js严格区分大小写
		
	3. 变量
		var 变量名;
		var 变量名 = 值;
		var 变量名 = 值, 变量名 = 值, ...;
		var 变量名1 = 变量名2 = 变量名3 = 值;

		命名规范:
		* 由数字, 字母, 下划线 和 $符号 组成
		* 不能以数字开头
		* 不能与关键字冲突
		* 严格区分大小写
		* 不能包含标点 和 空格

	4. 数据类型
		基本类型
			Bool, String, Number

		复合类型
			Array, Object

		特殊类型
			Null, undefined, function

	5. Bool	
		值: true 和 false
		严格区分大小写, 要求纯小写
		作用: 用于判断状态, 真假

	6. String
		格式: 
			单双引号
			反引号
		
		区别:
			单双引号
				不解析变量
				解析转义字符
				支持互插, 不支持自插
				不支持换行

			反引号
				解析变量  ${变量名}
				支持换行

		拼接:
			+ 

	7. Number
		整数 
							基数 		前缀
			二进制 			0-1 		0b
			八进制 			0-7 		0
			十进制 			0-9 		无
			十六进制 		0-9 a-f 	0x
		
			范围: -2^53 ~ 2^53

		小数
			科学计数法
			无穷数
				正无穷 Infinity
				负无穷 -Infinity
			范围: 
				5e-324 ~ 1.7 .... e308

			有效位数: 17位

			isFinite( )
			变量.toFixed( 保留位数 )

		NaN 
			作用: Not a Number  不是一个数

			NaN 与任何数字运算, 结果都是NaN
			NaN 与任何数字比较, 结果都不相等, 包括自己

			判断是否为NaN:  isNaN( ) 

	8. null 空
		代表什么都没有
			
		作用: 主要用于初始化值 或 清空值
		null 是关键字

	9. undefined 未定义
		以下几种情况值为 undefined
		* 直接赋值
		* 只定义变量, 不赋值
		
		null 和  undefined的区别
			null == undefined

			null是关键字, undefined 不是关键字

	10. 数据类型转换
		Bool转换
			Boolean( ) 强制转为 bool型
			以下几种情况会转为false
			* 0 或 0.0
			* -0
			* null
			* 空字符串
			* undefined
			* NaN

		String 转换
			String( ) 强制转为 string 型

		Number 转换
			Number() 	
			parseInt()
				以数字开头的字符串, 仅保留前面的数字
				以非数字开头的字符串, 全部转为NaN
				二进制   => 0
				八进制   => 保留
				十六进制 => 转为十进制

			parseFloat() 	
				同上
				识别科学计数法
				二进制   => 0
				八进制   => 保留
				十六进制 => 0
				只有第一个小数点有效
