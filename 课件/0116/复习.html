1. 函数参数
		实参: 调用函数给的参数
		形参: 定义函数给的参数

		实参个数 = 形参个数
		实参个数 > 形参个数   多余实参, 被抛弃
		实参个数 < 形参个数   多余形参, 当成undefined
										采用默认值

		实参个数 不确定时???
			形参就不写了, 通过 arguments 获取所有的实参

	2. 作用域
		全局变量:
			在函数外部用var声明的变量
			在函数内/外不用var 声明的变量 (隐式全局变量)

		局部变量
			在函数内部用var声明的变量

		作用域链
			对下透明: 下面的用上面的
			对上封闭: 上面不能用下面的

		访问顺序:
			优先 当前作用域
			其次 向上一级作用域查找
			最终 都找不到, 则报错

		作用域是在函数声明就已经定义好了, 不是在调用函数时定义

	3. 标签操作
		获取标签对象
			document.getElementById( )

		设置/获取标签 style属性
			标签对象.style.属性名 = 属性值

		设置/获取标签 正文内容 	
			标签对象.innerHTML = 值

	4. 定时器
		setTimeout(代码, 毫秒) 		倒计定时器	
		clearTimeout()

		setInterval(代码, 毫秒) 	循环定时器
		clearInterval()

	5. 对象
		构造方式
			var 对象名 = new Object()

		JSON方式
			var 对象名 = {
				属性名: 属性值,
				属性名: 属性值,
				...
				属性名: 属性值,
				方法名: function(){ 代码块 },
				方法名: function(){ 代码块 },
				...
				方法名: function(){ 代码块 }
			}

		自定义构造
			function 函数名(){
				this.属性名1 = 属性值1;
				this.属性名2 = 属性值2;
				...


				this.方法名 = function(){
					代码块
				}
			}

			this 当前对象, 用在function内

		匿名构造
			var 对象名 = new function(){
				代码块
			}
