1. 分支
		if 条件: true 环境

		if 条件:
			true环境 (代码块)

		if 条件:
			true环境
		else:
			false环境

		if 条件1:
			true环境1
		elif 条件2:
			true环境2
		elif 条件3:
			true环境3
		...


		if 条件1:
			true环境1
			if 条件2:
				true环境2
				if 条件3:
					true环境3
					...

	2. 循环
		while循环
			while 条件:
				true环境 (代码块)


			条件成立: 则进入代码块, 当代码块执行完毕, 再回到条件继续判断
			条件不成立: 则进入循环外面
			

			while 条件: true环境

			while 条件:
				true环境
			else:
				false环境


			特殊状态:
				条件 永远成立, 属于无限循环, 死循环

		for循环
			for 变量 in 容器:
				代码块

			变量 每一次循环, 只能获取容器中一个数据

			for 变量 in 数字区间:
				代码块

			数字区间:
				range(start, stop, step)


			for 变量 in 容器:
				代码块
			else:
				代码块
