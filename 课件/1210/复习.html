1. 函数
		将代码组织在一起, 形成一个完整的功能

		降低冗余率, 提高复用性
		提高开发效率
		便于代码维护

		内置函数
		自定义函数

		def 函数名( 参数 ):
			函数体

		函数不调用, 不执行
		调用函数的位置  必须在  定义函数的位置的后面
		先调用函数, 再执行函数体, 执行完了, 自动返回到 调用函数的地方 

	2. 函数名
		命名规范: 由数字, 字母和下划线组成, 但不能以数字开头
		区分大小写, 推荐小写
		命名形式:
			* 小写单词
			* 下划线
		函数名要有意义: 动词+名词
		函数名可以重复, 后面的会覆盖前面的. 尽量不要重复

	3. 返回值
		关键字: return
		作用:
			将函数的值 返回到调用函数的地方
			提前结束的函数  (默认情况下, 整个函数体执行完才会回到调用函数的地方)
		返回值:
			能返回任意类型的值
			返回多个值:
				return 值1, 值2, 值3, ... 		以元组形式返回
				return [值1, 值2 ...]
				 	   容器

		没有return , 就没有返回值.
		没有return, 整个函数体执行完就会回到调用函数的地方

	4. 参数
		形参: 在定义函数给的参数 parameter
		实参: 在调用函数给的参数 argument 

		参数传递:
			传值: 不可变类型, number, string, tuple, set
			传址: 可变类型, list, dict

		参数类型:
			必须参数:   按照顺序进行传输
			关键字参数: 顺序随意, 但需要提前声明参数名
			默认参数: 实参个数不足时, 形参即可采用默认参数
			不定长参数: 
				在形参前面加*,  以元组形式接收没人要的实参
				在形参前面加**, 以字典形式接收没人要的关键字实参
				单独加*,  在*后面的参数必须接收以关键字传入的实参  (实参的参数名固定)

		匿名函数:
			没有函数名的函数, 通过lambda 来定义函数
			作用: 
				用于简单的函数, 不需要取名

			格式:
				lambda 形参 : 返回值
