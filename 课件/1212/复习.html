1. 列表
		由一堆数据组成有序序列
		变量名 = [值1, 值2, ...]
		访问, 修改, 删除, 插入操作. 通过索引来操作
		
		访问, 修改, 删除 索引存在可以正常操作, 索引不存在, 都会报错
		插入, 索引已存在, 会影响原来的索引
			  索引不存在, 插入到最后

		* 重复
		+ 合并
		[:] 分片
		in  not in

		while 主要通过索引来遍历列表
		for   可通过索引, 可通过值来遍历列表

		列表中嵌套了另一个列表: 嵌套列表

		推导式: 	
			[ 变量操作 for 变量 in 列表 ]
			[ 变量操作 for 变量 in 列表 if 条件 ]
			[ 变量操作(T) if 条件 else 变量操作(F) for 变量 in 列表 ]
			[ 变量操作 for 变量 in 列表 for 变量 in 列表 ]
			[ 变量操作 for 变量 in 列表 for 变量 in 列表 if 条件 ]
			[ 变量操作(T) if 条件 else 变量操作(F) for 变量 in 列表 for 变量 in 列表 ]

	2. 元组
		一组有序序列, 不能修改数据. 元组属于不可变数据类型

		变量名 = (值1, 值2, ...)

		元组可以正常访问, 通过索引来操作
		但是, 不能修改元组, 不能删除元组.
		保护数据的作用


		* 重复
		+ 合并
		[] 分片
		in 成员检测

		while, for 来遍历
