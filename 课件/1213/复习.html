1. 字典
		由键值对 组成的可变类型
		键:
			支持数字, 字母. 推荐使用字母
			必须唯一.  如果重复, 后面的会覆盖前面的
			不能为空
		值: 任意类型

		格式: {键:值, 键:值, ...}

		基本操作:
			访问: 通过键 来访问
			修改: 通过已有的键 来修改
			插入: 通过没有的键 来插入
			删除: 通过已有的键 来删除
			成员检测: in

		for 变量 in 字典:
			变量获取键

		for 键, 值 in 字典.items():
			
		for 键 in 字典.keys():

		for 值 in 字典.values():

	2. 集合
		一个不会重复的无序序列, 是可变类型

		主要作用:
			去重
			关系测试


		格式: {值1, 值2, ...}
			  set(  容器   )

		基本操作
			差集 	-
			交集	 	&
			并集 	|
			对称差集		^
			成员检测 	in

		高级操作
			添加元素
				集合.add( 值 )
				集合.update( 容器 )

			删除元素
				集合.remove() 	
				集合.discard()
			
		遍历:
			for 变量 in 集合:


		字典和集合的推导式
			{ 变量操作 for 变量 in 字典.items()/集合 }
			{ 变量操作 for 变量 in 字典.items()/集合 if 条件}
			{ 变量操作 for 变量 in 字典.items()/集合 for 变量 in 字典.items()/集合  }
			{ 变量操作 for 变量 in 字典.items()/集合 for 变量 in 字典.items()/集合 if 条件 }
		
		冰冻集合 frozenset
			不可变类型
			一旦创建, 就无法再次修改,删除, 添加, 只能访问
			格式: frozenset( 可迭代对象 )

			遍历: 
				for 变量 in 冰冻集合:

			推导式:
				frozenset( 变量操作 for 变量 in 集合 )
				frozenset( 变量操作 for 变量 in 集合 if 条件 )
				...

	3. 迭代器
		可迭代对象(iterable): 只要能用 iter() 的对象都是可迭代对象
					string
					list
					tuple
					set
					dict
					range
					...

		迭代器(iterator): 将 可迭代对象里面的值 一个一个的获取出来, 不需要连续

		
		迭代器的使用:
			1. 创建迭代器	iter( 可迭代对象 )
			2. 一个一个获取值 next(迭代器)
		

	4. 高阶函数
		参数为 函数
		返回值为 函数
		满足其中之一, 就是一个高阶函数


		map(func, iterable)
		功能: 将可迭代对象中每一个对象  都使用一次函数
		参数:
			func: 函数(内置 或  自定义)
			iterable: 可迭代对象
		返回值:  对象
