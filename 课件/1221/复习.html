1. class
		class 类名():
			代码块


		命名规范
			类名:
				大驼峰命名法
				名词
				严格区分大小写

			方法名:
				动词

			属性名:
				由数字, 字母和下划线组成
				不能以数字开头

			方法名和属性名:
				一律小写, 多个单词之间用下划线隔开
				严格区分大小写

		类一旦定义, 就会占据内存
		类不实例化时, 不能使用

	2. object
		创建一个对象
			变量名 = 类名()

		使用属性
			对象名.属性名

		使用方法
			对象名.方法名()

		self
			当前对象
			只能在类内使用
			必须写在方法参数的第一个
			self 可以改名为其他单词, 不是关键字

		类 和 对象顺序
			先有类, 才会有对象

	3. 操作类
		类内:
			self.属性名 = 值
			
		类外
			对象名.属性名 = 值

		属性名已存在, 则为修改操作
		属性名不存在, 则为新增操作

	4. 专属方法
		__init__
			构造方法
			触发条件: 实例化类时
			作用: 初始化属性

		__del__
			析构方法
			触发条件: 当对象被销毁时
						1) 程序自然结束时
						2) 对象被覆盖时
						3) 对象被删除时
			作用: 善后

	5. OOP
		封装性
		继承性
		多态性

	6. 封装性
		将属性和方法集中在一起, 形成一个不可分割的类
		属性或方法有的公开, 有的私有

		私有化:
			在属性或方法前面加 两个下划线
			私有化的属性 和 方法 都只能在类内使用

			__属性名
			def __方法名():
